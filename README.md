
# PERN Stack Todlist (prisma )

A simple todo list with prisma orm and postgres. 

Features:



## Features

- Login/Registration
- Crud Todos
- Search Todos matching description/data/deadlines


## Tech Stack

Node, Express, Postgres, Prisma


## Running Tests

To run tests, run the following command

```bash
  npm install 
  npm run server 
```
